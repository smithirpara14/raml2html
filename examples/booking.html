Create Booking

Create a new booking with the provided details.

/create post

post

Create a new booking with the provided details.

post /create

Create a new booking with the provided details.

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • Account_Id: required(string)

    Salesforce Account Id

    Example:

    001XXXXXXXXX
  • Consultant_Email: required(string)

    Email of the consultant

    Example:

    consultant@company.com
  • Quote_Reference: required(string)

    Quote reference number

    Example:

    REF-14534
  • Passengers: required(integer)

    Number of passengers

    Example:

    4
  • TXN_Currency: required(string)

    Transaction currency

    Example:

    GBP
  • Travel_Start_Date: required(string - pattern: ^\d{4}-\d{2}-\d{2}$)

    Start date of travel

    Example:

    2024-10-17
  • Travel_End_Date: required(string - pattern: ^\d{4}-\d{2}-\d{2}$)

    End date of travel

    Example:

    2024-10-25
  • Client_First_Name: required(string)

    First name of the client

    Example:

    Jane
  • Client_Last_Name: required(string)

    Last name of the client

    Example:

    Doe
  • Client_Email: required(string)

    Email of the client

    Example:

    jane.doe@example.com
  • Itinerary_Link: (string)

    Link to the itinerary (optional)

    Example:

    test.itinerarylink.com

Example:

create-booking-request:

{
    "Account_Id": "001XXXXXXXXX",
    "Consultant_Email": "consultant@company.com",
    "Quote_Reference": "TEST-12345",
    "Passengers": 3,
    "TXN_Currency": "USD",
    "Travel_Start_Date": "2024-12-01",
    "Travel_End_Date": "2024-12-10",
    "Client_First_Name": "Jane",
    "Client_Last_Name": "Doe",
    "Client_Email": "jane.doe@example.com",
    "Itinerary_Link": "test.itinerarylink.com"
}

HTTP status code 200

Body

Media type: application/json

Type: object

Examples:

successful_response:

{
  "success": true,
  "message": "Booking has successfully been created.",
  "Booking_Id": "BKG-00101",
  "Proceeding": true,
  "Validity_Date": "2024-10-29",
  "TXN_Currency": "USD",
  "Confirmed": false,
  "Deposit_Due_Date": "2024-10-24",
  "Balance_Due_Date": "2026-07-29",
  "Booking_Summary_Link": "https://go.roampayments.com/s/booking?id=ABC123XYZ789",
  "Quote_Reference": "REF-12345",
  "Client_Email": "jane.doe@example.com",
  "Client_First_Name": "Jane",
  "Client_Last_Name": "Doe",
  "Passengers": 4,
  "Travel_Start_Date": "2024-12-01",
  "Travel_End_Date": "2024-12-10",
  "Itinerary_Link": null
}

error_response:

{
  "success": false,
  "message": "The account id listed as a parameter in the request is not related to an account in Roam."
}
