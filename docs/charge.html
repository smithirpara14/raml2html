Create Charge

Creates a new charge for the specified booking.

/create post

post

Creates a new charge for the specified booking.

post /create

Creates a new charge for the specified booking.

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • Account_Id: required(string)

    ID of the account related to the booking

    Example:

    001XXXXXXXXX
  • Consultant_Email: required(string)

    Email of the consultant handling the booking

    Example:

    consultant@company.com
  • Booking_Id: required(string)

    Unique ID of the booking

    Example:

    BKG-00101
  • BSE_Amount: required(number)

    Amount in base currency with 2 decimal precision

    Example:

    1500
  • BSE_Currency: required(string)

    Currency code, 3 characters

    Example:

    ZAR
  • Due_Date: required(string - pattern: ^\d{4}-\d{2}-\d{2}$)

    Due date for the charge

    Example:

    2024-02-09
  • Type: required(one of Deposit, Balance, Credit, Refund)

    Type of the charge

    Example:

    Balance
  • Beneficiary_1_Id: required(string)

    ID of the first beneficiary

    Example:

    BENIF001
  • Beneficiary_2_Id: (string)

    ID of the second beneficiary

    Example:

    BENIF002
  • Beneficiary_3_Id: (string)

    ID of the third beneficiary

    Example:

    BENIF003
  • Supplier_2_Amount: (number)

    Amount for the second supplier with 2 decimal precision

    Example:

    150
  • Supplier_3_Amount: (number)

    Amount for the third supplier with 2 decimal precision

    Example:

    50
  • API_Transaction_Id: (string)

    Unique ID for API tracking purposes

    Example:

    API-12345-TRANS

Example:

create-charge-request:

{
  "Account_Id": "001XXXXXXXXX",
  "Consultant_Email": "consultant@company.com",
  "Booking_Id": "BKG-00101",
  "BSE_Amount": 1500,
  "BSE_Currency": "ZAR",
  "Due_Date": "2024-02-09",
  "Type": "Balance",
  "Beneficiary_1_Id": "BENIF001",
  "Beneficiary_2_Id": "BENIF002",
  "Beneficiary_3_Id": "BENIF003",
  "Supplier_2_Amount": 150,
  "Supplier_3_Amount": 50,
  "API_Transaction_Id": "API-12345-TRANS"
}

HTTP status code 200

Body

Media type: application/json

Type: object

Examples:

successful_response:

  "Type": "Balance",
